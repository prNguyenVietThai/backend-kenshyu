#Setup project

##Install docker

##Build and run project
```sh
cd backend-kenshyu
docker-compose up
```
##Setup Database
Open brower and access to address:
```sh
http://localhost:8000
```
Root account:
- username: root
- password: 123456

User account:
- username: devuser
- password: devpass

Create 'test_db' database and import data from file:
```sh
/backend-kenshyu/database/test_db.sql
```
